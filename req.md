Інтерактивна карта університету з пошуком аудиторій та навігацією
Створити веб- або мобільний сервіс, який дозволяє:
• знайти потрібну аудиторію чи корпус
• побачити розташування на карті
• прокласти маршрут між аудиторіями
• швидко орієнтуватися в університеті
Основні функціональні модулі
Карта університету (головний модуль)
На карті мають бути позначені:
• корпуси
• аудиторії
• кафедри
• бібліотека
• спортзал
• гуртожитки
• адміністративні відділи
Формат карти:
• Google Maps API
або
• власний план поверхів
Користувач повинен мати можливість:
• ввести номер аудиторії (наприклад, 215)
• знайти корпус
• знайти кафедру або підрозділ
Результат:
• підсвічування на карті
• коротка інформація
Обов’язкова функція:
• прокласти шлях від одного місця до іншого
Наприклад:
• від входу до аудиторії
• між корпусами
• від гуртожитку до бібліотеки
Можливі режими:
• пішки
• для людей з інвалідністю (безбар’єрні маршрути)
Адміністратор може:
• додавати нові аудиторії
• редагувати інформацію
• оновлювати карту
. Основні сторінки застосунку
1. Головна карта
2. Пошук аудиторії
3. Список корпусів
4. Навігація (маршрут)
5. Інформація про об’єкт
6. Адмін-панель (за потреби)
Рекомендований стек  
Frontend:
• HTML/CSS/JavaScript
або React/Vue
Карти:
• Leaflet.js + OpenStreetMap (безкоштовно)
Backend:
• Django / Flask
або Node.js
Database:
• PostgreSQL / MySQL / SQLite
Додаткові можливості
При натисканні на аудиторію:
• фото
• опис
• контакти кафедри
• графік роботи
Розширення
• QR-коди біля аудиторій
• голосовий пошук
• мобільна версія
• інтеграція з розкладом занять
• режим “першокурсник” (швидкі маршрути)
